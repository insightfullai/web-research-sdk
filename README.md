# Web Research SDK Workspace

Monorepo scaffold for the public Web Research SDK packages using `vp` and Yarn workspaces.

## Packages

- `@insightfull/web-research-sdk` in `packages/core`
- `@insightfull/web-research-sdk-react` in `packages/react`
- `@insightfull/web-research-sdk-shared` in `packages/shared` (private)

## Commands

```bash
vp install
vp check
vp run -r test
vp run -r build
vp run -r pack
```
